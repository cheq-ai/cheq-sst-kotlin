class Layer

A layer in storage that stores key-value pairs to persistent storage.

The storage layer is a simple key-value store that can be used to store data that needs to be included in every Sst.trackEvent call. The storage layer is stored in the application's data store, and can be accessed using the add, all, clear, contains, get, and remove methods.
